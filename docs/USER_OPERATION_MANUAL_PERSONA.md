## 系统概述

### 核心价值

CMS 自动化系统是一个 **AI 驱动的内容管理自动化平台**，帮助内容团队实现：

- **智能内容抓取**: 自动从健康网站抓取文章
- **AI 驱动校对**: 基于 450+ 条规则的智能校对
- **SEO 优化**: 自动生成关键词、Meta 描述和 FAQ
- **可视化审核**: Before/After 对比，用户完全掌控
- **一键发布**: 确认后直接发布到 WordPress

### 状态颜色对照表

| 状态 | 颜色 | 含义 | 用户操作 |
|------|------|------|----------|
| `pending` | 灰色 | 等待处理 | 等待系统自动处理 |
| `parsing` | 蓝色 | 解析中 | 等待 AI 解析完成 |
| `parsing_review` | 琥珀色 | 解析审核 | 审核解析结果 |
| `proofreading` | 紫色 | 校对中 | 等待 AI 校对完成 |
| `proofreading_review` | 橙色 | 校对审核 | **主要工作：审核校对** |
| `ready_to_publish` | 青色 | 待发布 | 确认发布 |
| `publishing` | 靛蓝色 | 发布中 | 等待发布完成 |
| `published` | 绿色 | 已发布 | 完成 |
| `failed` | 红色 | 失败 | 检查错误并重试 |

## 常见问题与解决方案

### Q1: 文章一直卡在 "解析中" 状态怎么办？

**问题**: 文章状态显示 "parsing" 超过 5 分钟没有变化。

**解决方案**:
1. 刷新页面查看最新状态
2. 检查 Pipeline 监控页面，查看是否有错误
3. 如果持续卡住，联系系统管理员检查 AI 服务状态
4. 可能需要重新触发解析

### Q2: 校对建议全部接受后，发现有问题怎么办？

**问题**: 批量接受了所有建议，但发现某些修改不合适。

**解决方案**:
1. 在校对审核页面，找到需要修改的问题
2. 点击 [❌ 拒绝] 取消之前的接受
3. 或点击 [手动编辑] 进行自定义修改
4. 重新确认后继续下一步

### Q3: SEO 关键词如何选择才最优？

**问题**: AI 推荐了 8 个关键词，不知道如何选择。

**建议**:
1. 优先选择与文章主题高度相关的关键词
2. 包含 1-2 个高搜索量的核心关键词
3. 添加 2-3 个长尾关键词增加覆盖
4. 避免选择太泛泛的通用词
5. 建议总数控制在 5-7 个

### Q4: FAQ Schema 应该选择哪个版本？

**问题**: 有简洁版、标准版、详细版三个选项。

**建议**:

- **标准版 (5个)**: 适合中等文章（1000-2000字）
- **详细版 (7个)**: 适合深度文章（>2000字）

### Q5: 系统显示空白或加载失败怎么办？

**解决方案**:

2. 检查网络连接是否正常
3. 清除浏览器缓存后重试
4. 如果问题持续，联系系统管理员
